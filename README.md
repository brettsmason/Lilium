Lilium
========

Lilium is a clean and elegant child theme for Stargazer – One of the most advanced themes to grace the WordPress universe. This child theme uses a contemporary selection of fonts as well as a bright modern colour palette to transform the look and feel of the theme. It comes packaged with additional backgrounds and header images to use with the WordPress custom background/custom header features.
